# ML Label

Bounding box annotation tool for macOS. Annotate a set of images and export a JSON file for model training in Create ML.

- Drag and drop image import
- Resizable bounding boxes
- Export annotations as a JSON file in Create ML compatible [format](https://developer.apple.com/documentation/createml/building-an-object-detector-data-source)
- Save and resume projects as .mlset files
